Instrucciones y comandos que uso en el video:

La IP tuya va a ser diferente, por eso pégala acá: IP


Para los dos:

1. Abrir VSCODE como administrador 
2. choco install mkcert
3. mkcert -install
4. mkcert -key-file key.pem -cert-file cert.pem localhost IP

COMANDO =  WINDOWS + R (TECLAS)

Para django:

1. pipenv install django-extensions Werkzeug pyOpenSSL
2. Incluir en apps: "django_extensions",
3. python manage.py runserver_plus IP:8000 --cert-file cert.pem --key-file key.pem
4. Agregar la IP a ALLOWED_HOSTS en setting.py

Para que funcione:

1. Abrir cualquier endpoint de la api y verificarle al navegador que es seguro: https://IP:8000/api/pacientes/

2. Abrir el FrontEnd: https://IP:3000/login

Por ahora, en cualquier dispositivo en el que se quiera abrir la app toca hacer eso
